# Examples

## 전체목록 예시

**질문:** 전체 일정표 보여줘.

**응답:**
```
구분 | 날짜 | 제목(거래처) | 청구주기 | 청구일/납부기한 | 금액 | 산정식/근거 | 확실여부 | 리마인드
계약 | 2026-03-15 | ABC Corp | - | - | - | 서명완료 | 확실 | 종료 4개월 전
청구 | 2026-02-10 | XYZ Ltd | 월간 | 02-10/02-20 | 1,000,000 | 계약서 참조 | 확실 | 3주 전
업무 | 2026-02-26 | 내부 회의 | - | - | - | - | 확실 | 오늘
```

## 특정 거래처 청구 질문 예시

**질문:** XYZ Ltd 다음 청구 언제야?

**응답:**
```
구분 | 날짜 | 제목(거래처) | 청구주기 | 청구일/납부기한 | 금액 | 산정식/근거 | 확실여부 | 리마인드
청구 | 2026-02-10 | XYZ Ltd | 월간 | 02-10/02-20 | 1,000,000 | 계약서 참조 | 확실 | 3주 전
```

## 업데이트 예시

**질문:** 청구일을 3월 5일로 변경해줘.

**응답:**
```
업데이트 완료: XYZ Ltd 청구일 2026-03-05로 수정됨.
```

```
계약 | 2026-06-01 | DEF Inc | - | - | - | 기간 만료 | 불확실 | 종료 4개월 전
```
