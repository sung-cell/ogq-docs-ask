# OGQ Docs Ask

흩어진 업무 문서를 한 곳에서 검색하는 AI 문서 검색 앱

## 개요

Google Drive, 로컬 폴더, MYBOX 동기화 폴더에 흩어진 계약서·견적서·업무 문서를 자연어로 검색하고 요약합니다.

**주요 기능:**
- 자연어 검색 — "KT 계약서 어디 있어?", "SKT 계약 종료 언제야?"
- 문서 요약 — 계약서·견적서 핵심 내용 3줄 요약
- 계약 정보 추출 — 기간, 금액, 수익배분, 조건 자동 추출
- 자동 인덱싱 — 신규 문서 감지 시 자동 인덱스 갱신

**지원 소스:**
- Google Drive
- 로컬 폴더
- MYBOX 동기화 폴더

## 설치 (macOS)

### DMG 설치

1. `OGQ Docs Ask-x.x.x.dmg` 파일 실행
2. 앱을 Applications 폴더로 드래그
3. Launchpad 또는 Applications 폴더에서 **OGQ Docs Ask** 실행

> **보안 경고가 뜨는 경우**  
> 시스템 설정 → 개인 정보 보호 및 보안 → "확인 없이 열기" 클릭

### 저장 경로

앱 데이터는 macOS userData 디렉토리에 저장됩니다:

```
~/Library/Application Support/OGQ Docs Ask/
├── .cache_text/     # 추출된 텍스트 캐시
├── .index/          # 문서 인덱스
│   ├── documents.json
│   └── contracts-meta.json
├── logs/            # 로그
└── window-state.json
```

## 사용법

### 앱 실행

1. OGQ Docs Ask 앱 실행
2. 질문 입력창에 자연어로 검색
3. Enter 또는 검색 버튼 클릭
4. 결과에서 파일 링크 클릭 시 문서 바로 열기

**예시 질문:**
```
SKT 계약서 어디 있어?
KT 계약서 요약해줘
비즈챗 단가 얼마야?
채팅+ 수익배분 몇 %야?
다음달 청구 일정 뭐야?
KT 계약 종료 언제야?
```

### 인덱스 생성

최초 실행 시 인덱스를 생성해야 검색이 빠릅니다:

1. 앱 실행 후 인덱스 상태 확인 (우측 상단)
2. 인덱스가 없으면 **인덱스 생성** 버튼 클릭
3. 인덱싱 완료 후 검색 사용

> 문서를 새로 추가한 경우 인덱스 재생성을 권장합니다.

## 검색 기능 상세

### 문서 찾기
문서 위치를 모를 때 키워드로 검색합니다.
```
SKT 계약서 어디 있어?
PG사 문서 찾아줘
```

### 문서 요약
계약서·견적서의 핵심 내용을 자동 요약합니다.
- 회사명, 서비스명, 문서 유형
- 계약 금액, 단가, 수익배분
- 계약 기간 (시작일 / 종료일)
- 수수료율

```
KT 계약서 요약해줘
비즈챗 견적서 정리해줘
```

### 계약 정보 추출
문서 내에서 특정 조건·조항을 찾습니다.
```
비즈챗 단가 얼마야?
채팅+ 수익배분 몇 %야?
SKT 계약서에 운영비 관련 조항 있어?
```

### 일정 조회
청구·계약 종료 일정을 확인합니다.
```
이번달 청구 총액 얼마야?
다음달 청구 일정 뭐야?
KT 계약 종료 언제야?
```

## 자동 인덱싱

앱이 실행 중일 때 문서 폴더를 감시하여 신규·수정 파일을 자동으로 인덱싱합니다.

- 파일 추가/수정/삭제 자동 감지
- 변경된 파일만 증분 인덱싱
- 임시 파일(`~$`, `.tmp`, `.DS_Store`) 자동 제외

## OCR 설정 (선택)

스캔본 PDF에서 텍스트를 추출하려면 OCR을 활성화합니다.

설정 파일: `config/ocr.config.json`

```json
{
  "enabled": false,
  "languages": ["kor", "eng"],
  "maxPages": 3,
  "minTextLength": 50
}
```

`"enabled": true`로 변경 후 앱을 재시작하면 스캔본 PDF도 검색됩니다.

> OCR 활성화 시 pdfjs-dist 및 tesseract.js 설치 필요.  
> 처리 속도가 일반 PDF보다 느립니다.
